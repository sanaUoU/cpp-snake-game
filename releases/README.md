# 🐍 贪吃蛇游戏 - 发布版本

## 📦 包含内容

### Mac 版本
- `mac/SnakeGame.app` - macOS 应用程序
- `mac/SnakeGame-Mac.zip` - 压缩包（用于分发）

**安装方法（Mac）：**
1. 解压 `SnakeGame-Mac.zip`
2. 双击 `SnakeGame.app` 即可运行

**注意：** 首次运行可能会提示"无法打开，因为无法验证开发者"
解决方法：右键点击 → 打开 → 仍要打开

### Windows 版本
- `windows/SnakeGame.exe` - Windows 可执行文件
- `windows/*.dll` - SFML 运行库
- `windows/README.txt` - 说明文件

**安装方法（Windows）：**
1. 解压所有文件到同一文件夹
2. 双击 `SnakeGame.exe` 运行

## 🎮 游戏操作

| 按键 | 功能 |
|------|------|
| ↑↓←→ | 控制方向 |
| W A S D | 也可使用 WASD |
| 空格 | 重新开始 |

## 🎯 游戏规则

- 吃红色食物增长
- 每个食物 +10 分
- 撞墙或撞到自己会失败
- 速度会越来越快

## 📊 技术信息

- **引擎**: SFML 3.0
- **语言**: C++20
- **文件大小**: ~5-10 MB
- **内存占用**: ~10-20 MB
